# DecodeLabs-Data-Analytics-Project2
# Project 2: Exploratory Data Analysis (EDA)

## Project Overview
This repository contains the deliverables for Week 2 (Project 2) of the DecodeLabs Industrial Training. The project focuses on conducting data profiling, identifying outliers, calculating central tendency measures, and formulating core business KPIs from a transactional dataset.

## Deliverables Included
1. **Dataset_Data_Analytics_Cleaned.xlsx**: The finalized Excel sheet with applied live formulas (`AVERAGE`, `MEDIAN`, `MODE`, `MIN`, `MAX`, `IFS`) and the newly created `Net-TotalProfit` column.
2. **EDA_Summary_Report.pdf**: A comprehensive analytical report outlining executive summaries, descriptive statistics tables, skewness interpretation, and business insights.
